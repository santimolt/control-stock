# PWA Icons

Para que la PWA sea instalable, necesitas crear dos archivos PNG:

- `icon-192.png` (192x192 pixels)
- `icon-512.png` (512x512 pixels)

## Opción 1: Usar los SVG incluidos

Puedes convertir los archivos `icon-192.svg` y `icon-512.svg` a PNG usando:

1. **Photoshop/GIMP**: Abrir el SVG y exportar como PNG
2. **Inkscape**: File > Export PNG Image
3. **Online**: https://cloudconvert.com/svg-to-png

## Opción 2: Crear tus propios iconos

Diseña tus iconos personalizados y guárdalos como:
- `icon-192.png` (192x192px)
- `icon-512.png` (512x512px)

Recomendaciones:
- Usa colores sólidos con buen contraste
- Incluye padding interno de ~10% para que se vea bien en todos los dispositivos
- Prueba en modo claro y oscuro
